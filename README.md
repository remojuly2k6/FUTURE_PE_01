# ✦ Remo Salon — Men's Grooming Website

A premium, responsive landing page for **Remo Salon**, Chennai's premier men's grooming destination. Built with pure HTML, CSS, and JavaScript — no frameworks, no dependencies.

## 🔗 Live Demo

> Deploy with GitHub Pages — see instructions below.

## 📁 Project Structure

```
remo-salon/
├── index.html       # Main HTML structure
├── styles.css       # All styling (dark theme, animations, responsive)
├── main.js          # Interactions, scroll effects, animations
└── README.md        # Project documentation
```

## ✨ Features

- 🎨 **Dark premium design** with gold (`#D4AF37`) accents
- 📱 **Fully responsive** — works on mobile, tablet, and desktop
- 🌀 **Smooth animations** — scroll reveal, particle effects, counter animation
- 🃏 **Card tilt effect** on hover (service & feature cards)
- 🍔 **Hamburger menu** for mobile navigation
- ⬆️ **Back-to-top button**
- ✅ **SEO optimized** — meta tags, Open Graph, semantic HTML

## 🚀 How to Run Locally

1. Open a terminal in the project folder
2. Run the built-in Python server:

```bash
python -m http.server 8080
```

3. Open your browser and go to: **http://localhost:8080**

> **Note:** The terminal will show `Serving HTTP on :: port 8080` — this is normal. Keep it running and open the URL in your browser.

## 🌐 Deploy with GitHub Pages

1. Push this project to a GitHub repository
2. Go to your repo → **Settings → Pages**
3. Under **Source**, select `main` branch and `/ (root)` folder
4. Click **Save**
5. Your site will be live at: `https://<your-username>.github.io/<repo-name>/`

## 📞 Contact & Booking

Update the phone number in `index.html` (line 380) to your actual number:

```html
<a href="tel:+919999999999" ...>
```
